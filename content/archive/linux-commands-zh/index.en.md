Move a file:

```
mv /original_directory /target_directory
```

Run silently in the background:

```
nohup command &
```

Read the output of a silent run:

```
cat nohup.out
```

List processes:

```
ps aux | grep process_name
```

Kill a process:

```
kill pid
```

Show file/folder permissions:

```
ls -l
```

Extract a tarball:

```
tar xvfz something.tar.gz
```

```
tar xvf something.tar
```
