Monthly cost: about 40 RMB.

1. Buy a domain: [https://sg.godaddy.com/](https://sg.godaddy.com/)

2. Buy a VPS: [https://my.vultr.com/](https://my.vultr.com/)

   In general, pick a VPS in the **Singapore, Japan, Canada, or US** regions running **CentOS 7.0**, around USD 5/month.

3. Install Baota Panel on the server:

```
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
```

4. Install WordPress:

https://blog.naibabiji.com/an-zhuang-wordpress
