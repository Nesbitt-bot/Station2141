Monthly cost: about 7 dollars

First, buy your own domain at [https://sg.godaddy.com/](https://sg.godaddy.com/)

Second, you should own a VPS on [https://my.vultr.com/](https://my.vultr.com/) . Remember to chose **Centos 7.0** as your default system.

Third, install baota on your VPS, open terminal and paste commend:

```
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
```

Then, install wordpress on baota by the following instructions. (Open google translate if you are not Chinese)

> [WordPress教程，安装到使用0基础建站教学（2020/12月更新）](https://blog.naibabiji.com/an-zhuang-wordpress)
